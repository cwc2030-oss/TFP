# Terra Firma Partners — Project Instructions

## Overview
Terra Firma is a hunting/land intelligence platform focused on Missouri. Business model centers on $19 parcel unlocks and $99/yr Pro subscriptions.

## Active Product SKUs (April 2026)
- **$19 Parcel Unlock** — one-time purchase via `/api/parcels/purchase`, unlocks parcel for download. `/intel` page shows "Hunt Report" as the downloadable PDF artifact.
- **$49 Land Intelligence Report** — one-time order created via `/map` page, uses `land_report` productType. Uses `PRICES` in `/api/orders/route.ts`, PRODUCT_INFO in `/api/checkout/route.ts`.
- **$99/yr Pro subscription** — recurring via `/pricing` + `/api/stripe/checkout`.

## Removed Products
- **$149 Hunt Intelligence Report** — fully removed April 24, 2026. `hunt_report` productType no longer exists. Do NOT reintroduce without user approval.

## Architecture Notes
- Stripe is in LIVE mode — never create/modify Stripe products or prices.
- `generateLandPdfDirect` in `lib/report-generators.ts` powers $49 land PDF. `generateHuntPdfDirect` was deleted.
- `/api/parcel-hunt-file/route.ts` is still used for the $19 parcel unlock flow — KEEP IT.
- `/intel` page references to "Hunt Report" refer to the downloadable PDF unlocked by the $19 parcel purchase, NOT the removed $149 product.
- `STRIPE_HUNT_PLAN_PRICE_ID` env var is the $19 parcel unlock Stripe Price ID — KEEP IT.
